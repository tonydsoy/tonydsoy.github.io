This is more of a list of really good software that I use but sometimes it isn't really obvious what tool you'll need for the occasion.
so, this is my list of tools that I use regularly for development and stuff.

Visual Studio Code / VSCodium

VSC is my preffered program for coding anything. It has support for almost every programming language and it is VERY customizable
I actually wrote most of this website in vscode, so good on them them for making a program I use pretty much every weekend.

GIMP

GIMP is a free photo editor that I personally used to make some of the icons on this website.
Its a little hard to learn but with some practice you'll get used to it eventually.
