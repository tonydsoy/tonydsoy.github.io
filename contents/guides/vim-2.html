~~ ADVANCED VIM ~~

so you might understand how to use vim like a regular text editor right? but what if you want to go even faster?

this "advanced" guide will mostly just be about keybinds because thats all you really need to write faster ¯\_(ツ)_/¯

Selection

Key/Combo Mode Action
v Normal Enter visual mode, moving cursor will now select text.
y Visual Copy (yank) text to paste later.
x Visual Cut text to paste later
p Normal Paste text
> Visual Shift text right
< Visual Shift text left

Navigation

Key/Combo Mode Action
j Normal Move cursor up
k Normal Move cursor down
h Normal Move cursor left
l Normal Move cursor right
$ Normal Move cursor to end of line
^ Normal Move cursor to start of line (ignore whitespace)
Shift + Arrow Right Normal Move cursor right across one word
Shift + Arrow Left Normal Move cursor left across one word

Macros

Key/Combo Mode Action
qX Normal start recording a macro, X can be replaced with any letter on your keyboard
q Normal stop recording a macro.
@X Normal play a macro! you must have recorded it beforehand, replace X with macro key
