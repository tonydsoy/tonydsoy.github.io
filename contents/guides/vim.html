~~ VIM TUTORIAL ~~

How does one use vim? How does one exit vim? This little guide will tell you pretty much all you'll need to know about vim

Start by opening a file with the command vim



You'll see something that looks about like this. Just typing will not do anything at this point.

You have to go into INSERT mode by pressing i
Then you can type and move your cursor around like a regular text editor.



after you're done writing whatever, press esc to leave INSERT mode.
There are a few possible routes from here. you can save your file using :w (filename) or save and quit using :wq (filename)
If you started VIM whilst passing a file vim /path/to/file you can shorten your exit to just :w or :wq

TIPS

To those who came here to see me solve the crazy issue of "how do i exit vim?"
press esc then type :q! this will always exit you from vim without stopping you for any unsaved changes.

Extra features

VIM Isn't just a text editor. its also a file browser!
Use :Ex to view a simple file browser. and press enter on a file to open it.
If you want to open a new file in the directory you are in. Type :o (filename)

Prefix these commands with V or S to open them in split-view! V for vertical and S for horizontal.
e.g. :Vex :Vo (filename) :Sex :So (filename)
